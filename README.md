# 🎨 Color Palette Generator
A responsive web application that generates beautiful color palettes instantly. This project helps designers and developers explore color combinations and copy color codes with ease.

## 🚀 Live Demo

🔗 https://sonu1537.github.io/color-palette-generator/

## 📌 Features

- 🎨 Generate random color palettes
- 📋 Copy HEX color codes with one click
- ⚡ Fast and responsive interface
- 💻 Works on desktop and mobile devices
- 🎯 Simple and user-friendly design

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

- Responsive Web Design
